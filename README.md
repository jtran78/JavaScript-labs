# JavaScript and node.js Labs

A collection of lab exercises and small projects demonstrating **backend JavaScript development** using Node.js. These labs cover:

- Core JavaScript concepts for server-side programming  
- Asynchronous event-driven programming with callbacks and streams  
- Simple RESTful API development using Node.js built-in modules  
- Command-line argument parsing and input validation
